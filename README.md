# Finance Tracker App

A simple Finance Tracker web application built using HTML, CSS, and JavaScript. It helps users track their expenses by category and visualize spending using a pie chart.

## Features
- Add expense with amount and category
- Add custom category option
- Stores data using localStorage (data persists after refresh)
- Displays all expenses in a table
- Visualizes expenses using a pie chart (Chart.js)
- Clear all saved data option

- HTML
- CSS

- Chart.js
- localStorage API
